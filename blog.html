Portfolio Creation

James Shand, written on 30/11/22

During the creation of my portfolio page, I've learnt many new things along the way. Currently utilising html, css, and javascript to create a simple yet effective website without depending on abstract frameworks. I feel that learing the vanilla css and javascript before utilising tools like frameworks and libraries allows for a development of a solid foundation to then add on the skills and abstract understanding on top.
Having previously attempted to learn web devlopment by directly using tools like NextJS and React, I've come to theconclusion that shortcuts, in this case, wasn't the way to go. In the foreseable future, I see myself utilising frameworks like Selte and Kit to create highly interactive web apps. Svelte being a front-end web-framework, and SvelteKit being the serverside-rendering framework, acting as a backend to render, and fetch data from internal and external databases and api's.
